How to deploy to GitHub Pages (quick):
1. On GitHub, create a new public repository named: Mukulworld25.github.io
2. Upload all files from this ZIP root into the repository (you can drag-and-drop on GitHub web UI).
3. Commit and wait ~1 minute. Your site will be available at: https://Mukulworld25.github.io

How to update profile quickly:
- Edit data/profile.json and change fields (summary, skills, experience).
- Commit & push — GitHub Pages will auto-update the site.

Replacing the profile photo:
- Create a cleaned/cropped JPG named profile.jpg (square, ~600x600 px), and upload it to assets/img/profile.jpg (overwrite).
